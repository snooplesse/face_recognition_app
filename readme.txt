FACE RECOGNITION ATTENDANCE SYSTEM
===================================

QUICK START:
1. Extract all files to a folder
2. Run FaceAttendance.exe
3. If you get an error, install: https://aka.ms/vs/17/release/vc_redist.x64.exe

DEFAULT ADMIN LOGIN:
Username: admin
Password: 1234

USAGE:
1. Register Face - Anyone can register
2. Attendance (Scan) - Take attendance
3. Admin Login - View reports and train model

NOTE: After registering users, admin must click "Train Model" before attendance works!
